# Contribuir

Gracias por querer mejorar Mi Despensa.

Principios del proyecto:

- Mobile-first.
- Privacidad por diseño.
- Sin publicidad ni rastreadores ocultos.
- Lógica transparente y auditable.
- No agregar APIs externas de IA como dependencia obligatoria.

Antes de enviar cambios:

```bash
npm run build
npx tsc --noEmit
```

Usa commits descriptivos en español cuando sea posible.
